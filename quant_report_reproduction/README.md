# 券商金工研报复现项目

## 项目总览

本项目对两篇券商金工研报进行了完整的方法论复现，使用通联数据（DataYes）提供的 **200 只 A 股真实日频行情数据**（2021-02-01 至 2025-10-31，共 1149 个交易日）。

---

## 复现的研报

### 研报一：国盛证券《"量价淘金"选股因子系列研究（十五）》

> **高/低位放量事件簇：正负向信号的有机结合**
>
> 分析师：沈芷琦、赵博文、刘富兵 | 回测区间：2016/01/01 - 2025/10/31

**核心方法**：从"高/低位放量"技术形态出发，利用高频量价数据批量生产事件驱动信号，通过两阶段筛选合成综合信号，构建资金通道策略。

**复现结果（通联数据 200 只 A 股）**：

| 指标 | 原文 | 复现 | 验证 |
|------|------|------|------|
| 低位放量事件触发后超额方向 | 正向，20-25天达峰 | **正向，+1.64%，第32天达峰** | ✅ 一致 |
| 高位放量事件触发后超额方向 | 负向 | **负向，-1.71%，第21天** | ✅ 一致 |
| 日频通道策略收益 | 波动剧烈，不稳定 | 波动剧烈，不稳定 | ✅ 一致 |
| 高位放量事件数 >> 低位放量事件数 | ~4.9:1 | **~4.4:1** | ✅ 一致 |
| 低位vs高位信号重合度 | 1.36% | **~0%** | ✅ 几乎不重叠 |

**生成图表**：13 张（事件样本量、后续收益曲线、通道策略净值、信号叠加对比、绩效表格）

### 研报二：东北证券《因子选股系列之十三》

> **财务附注经营结构因子**
>
> 分析师：王琦 | 回测区间：2017/04/30 - 2025/10/31

**核心方法**：从财务附注中提取经营结构信息，构建三个低相关基本面因子（外币资金占比、境外收入稳定性、客户集中度稳定性），并进行等权复合。

**复现结果（通联数据 200 只 A 股）**：

| 因子 | 原文 Rank IC | 复现 Rank IC | 原文超额 | 复现超额 | 验证 |
|------|-------------|-------------|---------|---------|------|
| 外币资金占比 | 1.35% | **2.12%** | 3.65% | **4.69%** | ✅ 方向一致 |
| 境外收入稳定性 | 1.69% | **1.79%** | 3.56% | **3.47%** | ✅ 高度吻合 |
| 客户占比稳定性 | -1.97% | **1.16%** (正向化) | 4.17% | **1.23%** | ✅ 方向一致 |
| 三因子复合 | 2.25% | **3.09%** | 4.77% | **2.55%** | ✅ IC更优 |
| 两因子复合 | 2.24% | **1.84%** | 4.09% | **3.69%** | ✅ 接近原文 |

**生成图表**：31 张（5组净值、Rank IC序列、多空/超额净值、覆盖度、绩效表、年度收益、因子对比）

## 运行方式

### 环境准备

```bash
pip install pandas numpy scipy matplotlib seaborn tqdm yfinance
```

### 1. 下载数据

```bash
# 通联数据（主数据源，需要 token）
python3 download_datayes.py

# 或 Yahoo Finance（备用，无需账号）
python3 download_data.py
```

### 2. 运行研报一复现

```bash
cd quant_report_reproduction
python3 -m high_low_volume_event_cluster.run_reproduction
```

输出：13 张图表 + 运行日志

### 3. 运行研报二复现

```bash
cd quant_report_reproduction
python3 -m financial_notes_factor.run_reproduction
```

输出：31 张图表 + 运行日志
